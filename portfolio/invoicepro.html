Development

The current v.3 of the site was built as iteration on the previous two versions, with improved features and covering more use cases. The site started as Java program, which had its drawbacks, then became a single user web application that could be used without the need of java on the machine and with HTML5 Storage as database for storing different clients and products. In the 3rd version there is a support for multiple users with different accounts and full billing managing system. Also I am currently working on a paid subscription model for the application.

Info

This project was developed for an accountant that needed a simple way to issue invoices for customers from another country which required item currency exchange, some math and field validation that his standard software didn't allow.
